维基百科CDN中文翻译

少哥准备让我做应用分发子系统，所以近期会先学习下相关知识。以前看英文资料太少，感觉看了一遍后就忘得差不多，最近也老想翻译点东西，感觉有助于自己的理解，也有利于之后可能忘记时的回顾，翻译也让人比较容易静下心来，嗯，不错。

原文来自英文维基百科CDN

Content delivery network     内容分发网络

CDN，content delivery network或者content distribution network，是一个通过互联网部署在多个数据中心的大型服务器分布式系统。CSN的目的是高可用性、高性能地为端用户提供内容。CDN提供着当今互联网很大一部分的内容，包裹web对象（文本，图象，脚本），可下载对象（媒体文件，软件，文档），应用程序（电子商务，门户网站），实时流媒体，点播流媒体，社交网络。

CDN运营商从内容提供者，像媒体公司、电子商务卖家处挣钱，通过分发他们的内容给他们的端用户观众。反过来，CDN付钱给ISP，运输公司，以及网络运营商，为了在他们的数据中心托管它的服务器。除了更好的性能和可用性，CDN也卸载了直接从内容提供商的原始基础设施提供来的流量，使得内容提供商节约成本。此外，CDN通过使用大型分布式服务器基础设施去承受攻击流来为内容提供商提供一定程度上对DoS攻击的防护能力。虽然大多数的早期的CDN使用专有的服务器，这个服务器被CDN拥有、运营，但近来有一种使用P2P技术的复杂模型的趋势。在这个复杂模型中，同时使用专有服务器和其他合适的被对等用户拥有的计算机提供内容。

Operation    运营

大多数的CDN被作为一个应用服务提供商在互联网上运营着（也被认为是点播软件或者软件即服务）。越来越多的互联网网络拥有者建立他们自己的CDN去改善联网（on-net）内容的分发，减少通讯基础设施的需求，从内容客户产生收入。这可能包括了提供媒体流的使用权给互联网服务订阅者。一些更大型的软件公司，例如，微软，建立了他们自己的CDN串联他们自己的产品。例子包括Windows Azure CDN 和 Amazon CloudFront。

内容（潜在地有多个副本）可能存在于几个服务器上。当一个用户请求一个CDN主机名，DNS将决定出一个最优的服务器（基于位置、可用性、成本和其他度量），那个服务器将处理这个请求。

css的一些记录

这一周都在与css战斗，工作主要是模拟easyui，做一些样式出来，因为前端显示的网页很所都是多层嵌套，直接使用easyui很多会失效，有的由于表单的特殊性也没有合适的easyui样式套用。注意以下一些点。

  1. IE不识别颜色的rgb和hsl表示，尽量使用十六进制表示。
  2. colspan尽量在静态代码中定义，使用jQuery的attr始终没效果，colpsan和rowspan对table布局蛮重要的，用起来也很方便。

  3. 对于js单独文件嵌入问题，一般来说以下代码 

     (function(){
     $(window).ready(function(){
     //自己的代码
     })
 }).call(this)

    然后把js文件放入head中就没问题，但是那天遇到了将这段js代码反复执行了几次，不得要领，最后将js文件改成 

     (function(){
     //自己的代码
 }).call(this)

    把js文件放在body之后，就没有异常。现在也没明白原因。

  4. background-position这是针对图片本身而言，不是针对将图片放在网页中的相对位置。
  5. 悬浮效果可以通过设置div的position，一个是relative，一个是absolute。
  6. jQuery函数hover(function1, function2)，function1相当于mouseover函数，function2相当于mouseout函数。

响应式设计之媒体查询

媒体查询

  • 查询媒体设备的情况，加载相应的css文件
      <link media="link rel="stylesheet" media="screen and (orientation:portrait)" href="portraitscreen.css"/>
  • 媒体查询之间用分号分隔，没有后续表达式(例如：projection，后面没有任何特性/值得组合,意味着只要是projection就满足条件)
  • 可在css样式表中使用媒体查询，
      @media screen and (max-device-width: 400px) {
  h1 { color: green }
  }//屏幕小于等于400px的设备上，h1元素变绿
  • 可使用@import 按条件引入其他样式
      @import url("phone.css") screen and (max-width:360px);//最大宽度为360px的设备加载phone.css
    • 使用@import会增加HTTP请求，请谨慎使用
